### 包管理工具 Chocolatey

```
choco search 关键字
会搜索出所有包含关键字的软件包。

choco install 软件包名称
安装对应软件包

choco upgrade 软件包名称 
更新对应的软件包

choco uninstall 软件包名称
卸载对应的包

例如 安装nodejs 
choco install nodejs
```
https://chocolatey.org/
